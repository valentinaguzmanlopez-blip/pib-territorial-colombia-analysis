# PIB Territorial Colombia – Análisis Estructural y Sectorial

## Descripción del Proyecto

Este proyecto realiza un análisis estructural del Producto Interno Bruto (PIB) departamental de Colombia, con el objetivo de identificar patrones territoriales, concentración económica y dinámica sectorial a lo largo del tiempo.

El análisis combina limpieza, transformación y modelado de datos para evaluar la estructura productiva regional y posibles brechas territoriales.

---

## Objetivos

- Analizar la participación sectorial dentro del PIB departamental.
- Identificar concentración económica entre departamentos.
- Evaluar el crecimiento interanual por sector y territorio.
- Explorar posibles indicios de desigualdad regional.

---

## Metodología

1. Limpieza y estandarización de variables (Año y Valor).
2. Cálculo del PIB total por departamento y año.
3. Estimación de participación sectorial (%).
4. Ranking de departamentos por volumen económico.
5. Cálculo de crecimiento interanual por sector.
6. Medición de dispersión territorial mediante desviación estándar.

---

## Herramientas Utilizadas

- Python
- Pandas
- Matplotlib
- Google Colab

---

## Hallazgos Relevantes

- Se evidencia una alta concentración económica en determinados departamentos.
- El sector terciario presenta mayor participación relativa en la estructura productiva nacional.
- Existen variaciones significativas en las dinámicas de crecimiento entre sectores y territorios.
- La dispersión del PIB entre departamentos sugiere diferencias estructurales persistentes.

---

## Relevancia

Este análisis puede servir como insumo para estudios de desarrollo económico regional, formulación de políticas públicas territoriales y evaluación de estructura productiva.
